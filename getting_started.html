Getting Started

A quick guide that gets you started with Winnux 58

  1. Download our repository on GitHub: git clone https://github.com/The-UltimateGamer/GUIdo
  2. Navigate to the main directory: cd GUIdo
  3. "Boot" the program: python stub.py and have fun!
    Note: This requires Python 3 and an installation of the pygame library.
